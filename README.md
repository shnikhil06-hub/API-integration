# 🍕 Food Menu App

A modern and interactive food browsing web application built using **vanilla JavaScript**, designed to demonstrate API integration, dynamic UI rendering, and efficient data manipulation using array methods.

---

## 📌 Project Overview

This application fetches real-time recipe data from the **Spoonacular API** and displays it in a clean, responsive grid layout. Users can search, filter, sort, and interact with meals seamlessly.

---

## ✨ Features

### 🔍 Search & Filtering

* Search meals by name
* Filter by vegetarian option
* Filter meals by calorie range

### ⬆️ Sorting

* Sort by:

  * Name (A → Z / Z → A)
  * Popularity (likes)
  * Calories

### ❤️ Favorites System

* Add/remove meals to favorites
* Stored using **localStorage**
* Persists even after page reload

### 🌙 Dark Mode

* Toggle between light and dark themes
* User preference saved locally

### 📱 Responsive Design

* Fully optimized for mobile, tablet, and desktop

## 🧠 JavaScript Concepts Used

This project demonstrates the use of Array Higher-Order Functions:

* `filter()` → Used for search, vegetarian filter, and calorie filtering
* `sort()` → Used for sorting meals dynamically
* `map()` → Used to render UI components dynamically
