# 🍕 Food Menu App

A modern and interactive food browsing web application built using ** JavaScript**, designed to demonstrate API integration, dynamic UI rendering, and efficient data manipulation using array methods.

## 📌 Project Overview

This application fetches real-time recipe data from the **Spoonacular API** and displays it in a clean, responsive grid layout. Users can explore meals, apply filters, and interact with the interface seamlessly.

## ✨ Features

### 🔍 Search & Filtering

* Search meals by name
* Filter by category (e.g., Dessert, Main Course)
* Toggle vegetarian-only options

### ⬆️ Sorting

* Sort by:

  * Price
  * Calories
  * Alphabetical order

### ❤️ Favorites System

* Save favorite meals using **localStorage**
* Persist data across sessions

### 🌙 Dark Mode

* Toggle between light and dark themes
* Preference saved locally

### 📱 Responsive Design

* Fully optimized for mobile, tablet, and desktop

## 🧠 JavaScript Concepts Used

This project emphasizes the use of Array Higher-Order Functions:

* `filter()` → Handles search + category + veg filtering
* `sort()` → Controls ordering of results
* `map()` → Dynamically renders UI components
